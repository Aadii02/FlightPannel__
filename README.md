# Mission Control Window

This is a Python GUI application for a mission control window displaying three graphs for X, Y, and Z axes.

## Installation

1. Install Python 3.x
2. Install dependencies: `pip install -r requirements.txt`

## Usage
# there are two main python file 
# 1. main.py (uses tinktr and matplotlib) (slow)
# 2. main_dpg.py  (uses dearpygui) (pretty fast) 
Run `python main.py` to launch the application. 
Run 'python main_dpg.py' to launch the application
